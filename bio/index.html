Bio

Passionate about exploring the natural world and learning how to best live and grow within this natural environment in conjunction with the advancement of technology. David draws on the art, architecture, culture and people around him, embracing the unique differences and relishing in the many similarities we all share. Colour, line, and shape are the key aspects David explores to create vibrant and loud designs that are clean and sharp whilst maintaining a holistic ethos.

Tools of the Trade

  • Adobe Creative Suite
  • Figma
  • Visual Studio Code
  • HTML
  • CSS
  • SASS
  • Javascript
Familiar and confident with a wide range of Apps from the Adobe Create Suite. Any unkown aspects or skills are quickly learned via thorough research and/or online training modules. The main Apps used for the majoprity of projects are Illustrator, Photoshop, InDesign and Acrobat.
This is where each project starts after thorough research and analysis. Begining with ideation, wire frames, hieracrhy charts, user flows, compiling research notes and concept art. Each project is built from the ground up. Figma is excellent for generating prototypes for user testing and peer reviews.
After the design process is complete. Projects that require programing languages are compiled with Visual Studio Code. A one stop place for all things coding. With many different extensions, Visual Studio code is excellent. It links up directly using Git with online repositries such as Github for application and user testing. It is also excellent for online collabration of projects with peers or work colleagues.
